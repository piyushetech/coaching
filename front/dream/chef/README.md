# Chef — Recipe App UI

HTML/CSS design prototype matching the reference mockup (Home + Recipe Detail screens).

## Open

```
dream/chef/index.html
```

## Screens (side by side)

1. **Home** — Orange swirl header, Hello Jenny, search + filter, categories, Popular Recipes card, Top Chefs, bottom nav (Home · Discover · Favourite · Chat · Profile)

2. **Recipe Detail** — Burst Tomato Pasta hero, gallery thumbnails, author (Dianne Russell), description, cooking time / cuisine cards, ingredients list, **Watch Videos** button

## Mock data

`js/mock-data.js` — Jenny user, Burst Tomato Pasta, categories, chefs (Esther T., Jenny M., Jacob U., Bessi K.)

## Interactions

- Click a popular recipe card → scrolls to detail screen
- Search filters recipe cards
- Heart toggle on cards
- Gallery thumbnails swap hero image
- Read more / Read less on description

Design only — no backend.
